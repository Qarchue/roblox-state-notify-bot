# roblox-state-notify-bot

有一些玩家上線時都不會說一聲，導致很常錯過一起遊玩的機會，這非常難過，所以我使用 **Python 3** 來監視他們的狀態變化！

這是一個簡單的小專案，用於對指定玩家的狀態改變作出通知。更支援使用 Cookie 來查詢更詳盡的資料，例如玩家加入或離開的遊戲。

它還有設定語言的功能，對於那些想要使用不同語言接收通知的人來說，可以透過更改或新增 `language` 資料夾內的 JSON 檔案來自訂或修改顯示的語言內容。

_本文檔將引導您完成創建 Discord 機器人及使用其功能。_

---

## 特徵

- [x] 當玩家加入、離開、上線、下線、進入 Studio、離開 Studio 時，透過 Discord 通知。
- [x] 支援 Cookie 驗證，使用者提供 Cookie 後可以查詢玩家是否加入或離開某個遊戲。
- [x] 可自訂語言：複製 `language` 資料夾內的 JSON 檔案即可快速新增或修改顯示語言。

---

## 指令功能

| **指令名稱**   | **功能描述**                                                                                                                                                     |
|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `/sublist`     | 查看您目前訂閱的 Roblox 玩家列表。如果沒有訂閱任何玩家，會提醒如何開始訂閱。                                                                                     |
| `/sub`         | 訂閱或取消訂閱特定 Roblox 玩家。                                                                                                                                |
| `/setting`     | 更改機器人的語言設置，或更新 Roblox Cookie。如果提供有效的 Cookie，機器人將啟用進階查詢功能，例如監視好友是否進入/離開某個遊戲。                                      |

## JSON 設定說明

以下是需要建立的 JSON 檔案與其內容範例：

### 1. `settings.json`
**位置**：`configuration/settings.json`  
此檔案用於設定機器人的 Token 和預設語言。  
範例內容：
```json
{
    "TOKEN": "你的 Discord Bot Token",
    "Default_language": "zh-tw"
}
```
- `TOKEN`：填入您在 Discord 開發者平台取得的 Bot Token。
- `Default_language`：填入您希望的預設語言檔名稱（如 zh-TW）。

## 安裝依賴

確保已安裝 Python 3.8 或更新版本，然後執行以下命令來安裝依賴：

```sh
pip install -r requirements.txt
```

`requirements.txt` 檔案中包含了運行此機器人所需的所有 Python 套件。

## 啟動機器人

執行以下指令來啟動機器人：

```sh
python bot.py
```

---

## 注意事項

### 1. Roblox Cookie
提供有效的 `.ROBLOSECURITY` Cookie 可啟用進階功能，例如查詢好友是否加入或離開遊戲。如果 Cookie 無效，機器人會發出錯誤通知。

### 2. Discord 權限
確保機器人擁有以下基本權限：
- 發送訊息
- 嵌入訊息
- 使用應用程式指令

### 3. 語言設定
預設語言為 `zh-TW`，您可以新增或修改 `language` 資料夾中的檔案，並透過 `/setting` 指令更改語言。
